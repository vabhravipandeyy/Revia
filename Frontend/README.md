# Run the Revia frontend

This contains everything you need to run the frontend locally against the Revia backend.

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Set the frontend API base URL in `.env` or `.env.local`
   `VITE_API_BASE_URL="https://your-api-id.execute-api.ap-south-1.amazonaws.com/dev"`
3. Run the app:
   `npm run dev`
